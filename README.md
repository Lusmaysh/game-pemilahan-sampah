## ğŸ“– Cara Bermain
1.  **Mulai Game:** Klik tombol Start/Play pada menu utama.
2.  **Identifikasi Sampah:** Lihat objek sampah yang muncul di layar.
3.  **Pilah Sampah:**
    * Tarik (**Drag**) sampah tersebut.
    * Lepaskan (**Drop**) ke tong sampah yang sesuai warnanya/kategorinya (Misal: Hijau untuk Organik, Kuning untuk Anorganik).
4.  **Hindari Kesalahan:** Memasukkan ke tong yang salah akan mengurangi nyawa/skor.
5.  **Selesaikan Level:** Kumpulkan skor sebanyak mungkin.
